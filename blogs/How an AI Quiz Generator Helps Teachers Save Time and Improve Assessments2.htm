H2: The Challenges Teachers Face in Assessment Creation

Coming up with effective assessments takes quite a lot of time because a teacher must:

  • Design appropriate questions aligned with the learning outcomes.
  • Make sure that quizzes are balanced and objective.
  • Create different versions of quizzes to avoid cheating.
  • Review the quiz and determine learning problems.

Such duties are usually very difficult to fulfil, especially in a large classroom setting.
